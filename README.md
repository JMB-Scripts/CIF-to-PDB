# CIF-to-PDB
Convert in batch CIF format to a PDB 

The script will convert in batch mode CIF files in a folder to PDB in outputs_pdbs,
 
## Requirements
 - Python 3.x

## Python packages:
 - Open Babel

 conda install openbabel

## USAGE 

 python script_name.py path_to_CIFS_files_FOLDER/
